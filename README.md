# Hero Cafe - スマホアプリLPサイト

Hero Cafeアプリのランディングページサイトです。

## 構成

- **モバイルファースト**設計
- **767px以下**: モバイル・タブレット対応
- **768px以上**: デスクトップ対応（実装予定）

## ファイル構成

```

├── css/

├── js/
```
